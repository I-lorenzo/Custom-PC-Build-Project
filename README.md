# 🖥️ Custom PC Build Project

## 📌 Overview
Built a high-performance desktop PC focused on gaming, productivity, and system performance. This project demonstrates hands-on experience with hardware assembly, component compatibility, and system configuration.

---

## ⚙️ Hardware Specifications

| Component        | Model |
|-----------------|------|
| CPU             | Intel Core i9-10850K |
| Motherboard     | ASUS ROG Strix Z490-F Gaming |
| RAM             | 32GB DDR4 |
| GPU             | AMD Radeon RX 7800 XT |
| Power Supply    | Corsair RM850x (850W) |
| CPU Cooler      | Corsair H100i Liquid Cooler |
| Storage         | 1TB NVMe SSD + 250GB SSD |
| Case            | LANCOOL III |

---

## 🧱 Build Process

### 1. Component Selection
- Selected compatible components (Intel LGA1200 platform)
- Ensured adequate power delivery (850W PSU for GPU + CPU)
- Choose liquid cooling for improved thermal performance
- Included fast storage (NVMe SSD) for OS and applications

---

### 2. Assembly
- Installed CPU onto motherboard (LGA1200 socket)
- Mounted liquid cooler with proper thermal paste application
- Installed RAM in dual-channel configuration
- Mounted motherboard into case
- Installed PSU and completed cable management
- Installed GPU into PCIe slot
- Installed NVMe and SATA SSD storage drives

### 3. System Setup
- Booted into BIOS/UEFI
- Verified detection of all components
- Enabled XMP for RAM performance
- Installed Windows OS on NVMe SSD

---

### 4. Drivers & Configuration
- Installed chipset, LAN, and audio drivers
- Installed AMD GPU drivers (Adrenalin software)
- Performed full Windows updates

---

### 5. Testing & Optimization
- Monitored system temperatures under load
- Verified stability during normal use and stress testing
- Confirmed fast boot and load times using NVMe storage

- PC Hardware Assembly
- Storage Configuration (NVMe + SATA)
- BIOS/UEFI Configuration
- Thermal & Cooling Management
- Troubleshooting & System Optimization

Successfully built and configured a high-performance system with fast storage, stable performance, and efficient cooling suitable for gaming and productivity workloads.
